List of Builtin Filters

The below list demonstrates all the built in filters.

Builtin Filters operating on Integers, floats

  • Input: {{ num }}
  • Absolute: {{ num|abs }}
  • Float: {{ num|float }}
  • Integer: {{ num|int }}
  • Round: {{ num|round }}
  • String: {{ num|string }}
  • File Size Human readable format: {{ num|abs|int|filesizeformat }}

Builtin Filters operating on strings

  • Input: {{ name }}
  • Lower: {{ name|lower }}
  • Upper: {{ name|upper }}
  • Title: {{ name|title }}
  • Length: {{ name|length }}
  • Wordcount: {{ name|wordcount }}
  • Capitalize: {{ name|capitalize }}
  • Replace: {{ name|replace('Pattisapu', 'Priyatam')}}
  • Reverse: {{ name|reverse }}
  • Escape: {{ name|escape }}
  • Trim: {{ name|trim }}
  • Center: {{ name|center(width=100) }}
  • Indent: {{ name|indent }}
  • Format: {{ "My name is %s "|format(name) }}
  • Truncate: {{ name|truncate(length=6, killwords=True)}}
  • Wordwrap: {{ name|wordwrap(width=3)}}
  • List: {{ name|list }}
  • Pretty print: {{ name|pprint }}
  • Default values for undefined var: {{ var|default('I am Undefined') }}

Builtin Filters operating on sequences

  • Input: {{ mlist }}
  • Max: {{ mlist|max }}
  • Min: {{ mlist|min }}
  • Sum: {{ mlist|sum }}
  • First: {{ mlist|first }}
  • Last: {{ mlist|last }}
  • Length: {{ mlist|length }}

  • Random elem: {{ mlist|random }}
  • Sorted list: {{ mlist|sort }}
  • Unique values: {{ mlist|unique|list }}

  • Select attribute: {{ mlist2|selectattr('k1')|list }}
  • Reject attribute: {{ mlist2|rejectattr('k1')|list }}
  • Batch: {% for tb in mlist|batch(2) %} {{ tb }} and {% endfor %}
