List of Builtin Tests

A minimalist example which demonstrates the usage of filters. The below list demonstrates all the built in tests.

Builtin Filters operating on boolean

  • Input: {{ mbool }}
  • {{ "Variable is boolean" if mbool is boolean }}
  • {{ "Value is false" if mbool is false }}
  • {{ "Value is not true" if mbool is not true }}

Builtin tests operating on Integers, floats

  • {{ "Var defined" if var is defined else "Var is not defined" }}
  • {{ "%s is not divisible by 3"|format(num) if num is not divisibleby(3) }}
  • {{ "%s is not even"|format(num) if num is not even }}
  • {{ "%s is not odd"|format(num|abs|int) if num|abs|int is not odd }}
  • {{ "%s is equal to %s"|format(num, num) if num is eq(num) }}
  • {{ "%s is greater than or equal to 0"|format(num|abs) if num|abs is ge(0) }}
  • {{ "%s is greater than 0"|format(num|abs) if num|abs is gt(0) }}
  • {{ "%s is less than or equal to 1000"|format(num) if num is le(1000) }}
  • {{ "%s is less than 1000"|format(num) if num is lt(1000) }}
  • {{ "%s is not equal to 0"|format(num) if num is ne(0) }}
  • {{ "%s is not an int"|format(num) if num is not integer }}
  • {{ "%s is a float"|format(num) if num is float }}
  • {{ "%s is a number"|format(num) if num is number }}

Builtin tests operating on strings

  • Input: {{ name }}
  • {{ "%s is not lowercased"|format(name) if name is not lower }}
  • {{ "%s is not uppercased"|format(name) if name is not upper }}
  • {{ "%s is defined"|format(name) if name is defined}}

Builtin tests operating on sequences

  • Input: {{ mlist }}
  • {{ "3 is present in %s"|format(mlist) if 3 in mlist}}
  • {{ "%s is iterable"|format(mlist) if mlist is iterable}}
  • {{ "%s is a sequence"|format(mlist) if mlist is sequence}}

Builtin tests operating on dictionaries

  • Input: {{ mdict }}
  • {{ "%s is a mapping"|format(mdict) if mdict is mapping}}
  • {{ "%s is same as False"|format(mdict.k1) if mdict.k1 is sameas False}}

Builtin tests operating on functions

  • Input: {{ mfunc }}
  • {{ "%s is a function"|format(mfunc) if mfunc is callable}}
