List of Builtin Tests

The below list demonstrates all the built in tests.

Builtin Filters operating on boolean

  • Input: False
  • Variable is boolean
  • Value is false
  • Value is not true

Builtin tests operating on Integers, floats

  • Var is not defined
  • -76.3 is not divisible by 3
  • -76.3 is not even
  • 76 is not odd
  • -76.3 is equal to -76.3
  • 76.3 is greater than or equal to 0
  • 76.3 is greater than 0
  • -76.3 is less than or equal to 1000
  • -76.3 is less than 1000
  • -76.3 is not equal to 0
  • -76.3 is not an int
  • -76.3 is a float
  • -76.3 is a number

Builtin tests operating on strings

  • Input: nikhil Pattisapu
  • nikhil Pattisapu is not lowercased
  • nikhil Pattisapu is not uppercased
  • nikhil Pattisapu is defined

Builtin tests operating on sequences

  • Input: [3, 2, 4, 3]
  • 3 is present in [3, 2, 4, 3]
  • [3, 2, 4, 3] is iterable
  • [3, 2, 4, 3] is a sequence

Builtin tests operating on dictionaries

  • Input: {'k1': False, 'k2': True}
  • {'k1': False, 'k2': True} is a mapping
  • False is same as False
  • Sorted dict: [('k1', False), ('k2', True)]

Builtin tests operating on functions

  • Input: .identity_func at 0x7f324820ac80>
  • .identity_func at 0x7f324820ac80> is a function
